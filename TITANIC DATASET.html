def impute_age(cols):
    Age = cols.iloc[0]
    Pclass = cols.iloc[1]

    if pd.isnull(Age):

        if Pclass == 1:
            return 38
            
        elif Pclass == 2:
            return 29.5

        elif Pclass == 3:
            return 24.5
    else:
        return Age

CONCLUSION

In analyzing the Titanic dataset, several key observations emerged. Imputation of missing age values was conducted, predominantly relying on mean values stratified by passenger class. Notably, the cabin column, due to extensive missingness, was deemed unsuitable for analysis and thus dropped from consideration.

Exploration of passenger age distribution revealed a predominant concentration between the ages of 20 and 30. Concurrently, examination of fare distribution elucidated that a significant portion of passengers paid fares ranging from 0 to 100, with variability based on embarkation port and passenger class.

Further investigation into fare and embarkation port unveiled that passengers embarking from Southampton and Cherbourg tended to pay higher fares, albeit with comparable survival rates. This prompted speculation regarding the occupational roles of some Southampton passengers, potentially comprising ship personnel.

Correlation analysis underscored weak associations between age, fare, and familial companionship indicators such as the number of siblings/spouses and parents/children aboard. These insights contribute to a nuanced understanding of passenger demographics and socioeconomic dynamics aboard the Titanic, enriching historical comprehension of this seminal event.
